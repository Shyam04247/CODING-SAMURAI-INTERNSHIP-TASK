Movie Recommendation System & Sales Dashboard
Coding Samurai Business Analytics Internship

Projects Included:

1. Sales Performance Dashboard (Excel)

* Created a dashboard to analyze retail sales data.
* Calculated key metrics such as Total Revenue, Total Sales, and Average Order Value.
* Visualized monthly sales trends using line charts.
* Identified the top 3 products based on revenue.

2. Movie Recommendation System (Python)

* Built a simple movie recommendation system using the MovieLens dataset.
* Analyzed the most popular movies based on average ratings.
* Grouped movies by genre and visualized the number of movies per genre using a bar chart.
* Recommended top-rated movies for a specific genre.

Tools & Technologies Used:

* Excel
* Python
* Pandas
* Matplotlib

Dataset Used:
MovieLens Small Dataset
Source: https://grouplens.org/datasets/movielens/

How to Run the Python Project:

1. Download the MovieLens dataset from the link above.
2. Extract the files.
3. Place movies.csv and ratings.csv in the same folder as movie_recommendation.py.
4. Run the following command in terminal:

python movie_recommendation.py
